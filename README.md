# OneManager-cfworkerskv

1. Enter KV (Key Store), add a namespace (any English name will do).

2. Enter workers, create a new namespace, paste the code, and save.

3. In worker settings, bind the KV namespace, enter "OMKV" as the variable name, and select from the dropdown menu on the right.

4. Access to begin installation.

The familiar operation process requires no code modification; just click through.

Only the main functions are available: adding disks and refreshing the cache.

Unavailable: Encrypted directories, management operations, and uploads.
